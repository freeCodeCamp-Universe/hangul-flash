# 한글 Flash - README

**한글 Flash** is a browser-based flashcard and quiz app for learning to read Korean, from individual letters to full sentences.

## Key Features

**Four Learning Levels**: Content is organized into four progressive levels — Letters (자음/모음), Syllables, Words, and Sentences — so learners can build reading speed step by step.

**Practice Mode**: Players flip through flashcards and self-grade each card as Hard, OK, or Easy. A timer bar can be set to 3 s or 10 s to add pressure and train reading speed.

**Quiz Mode**: Players are shown a Korean character or phrase and must pick the correct romanization or meaning from four multiple-choice options, with immediate right/wrong feedback.

**Text-to-Speech**: Every card has a speaker button that plays native Korean pronunciation via the Web Speech API.

**Round Results**: After each round, a summary shows the Hard/OK/Easy breakdown and lists any cards marked Hard for quick review before moving on.

**Theory Tab**: A built-in reference explains how Hangul syllable blocks work, covering basic consonants, tense consonants, basic vowels, Y-vowels, compound vowels, and reading order rules.

**Configurable Round Size**: A slider lets players choose between 5 and 30 cards per round.

## Tech Stack

Vanilla HTML, CSS, and JavaScript. No build step or dependencies — open `index.html` directly in a browser.
