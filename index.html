How Hangul works

Korean is written in syllable blocks. Each block is made of 2–3 letters stacked together: an initial consonant (초성), a vowel (중성), and an optional final consonant (받침).

가 ㄱ + ㅏ ga
한 ㅎ + ㅏ + ㄴ han
글 ㄱ + ㅡ + ㄹ geul
닭 ㄷ + ㅏ + ㄹ + ㄱ dak — chicken
ㅇ is silent at the start of a syllable (acts as a placeholder when the syllable begins with a vowel), but sounds like ng at the end — e.g. 방 (bang).

Reading order

Within a syllable block, read left → right, top → bottom:

나 ㄴ + ㅏ na — initial + vertical vowel
고 ㄱ + ㅗ go — initial + horizontal vowel
산 ㅅ + ㅏ + ㄴ san — mountain
봐 ㅂ + ㅘ bwa — compound vowel
닭 ㄷ + ㅏ + ㄺ dak — double final consonant
읽 ㅇ + ㅣ + ㄺ ilk — to read (verb stem)
Vertical vowels (ㅏ ㅓ ㅣ ㅐ…) go to the right of the consonant: ㄴ + ㅏ → 나

Horizontal vowels (ㅗ ㅜ ㅡ…) go below the consonant: ㄱ + ㅗ → 고

Final consonant (받침) always sits at the bottom of the block: ㅅ + ㅏ + ㄴ → 산

Double final consonants (겹받침) — only one is usually pronounced. In 닭 (chicken), ㄺ = ㄹ + ㄱ, but you say dak.
